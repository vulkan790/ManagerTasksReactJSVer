# Task Manager (React)

Простое приложение для управления задачами, созданное на React. Позволяет добавлять, просматривать и удалять задачи с интуитивно понятным интерфейсом.

## Особенности

- **Добавление новых задач с валидацией пустого ввода**
- **Удаление задач в один клик**
- **Счётчик задач**
- **Интуитивно понятный UI/UX**

## Быстрый старт

### Предварительные требования
- **Node.js (версия 16 или выше)**
- **npm или yarn**

### Установка и запуск

1. **Клонируйте репозиторий**

```sh
git clone https://github.com/vulkan790/ManagerTasksReactJSVer.git
cd ManagerTasksReactJSVer
```

2. **Установите зависимости**
```sh
npm install
```
3. **Запустите в режиме разработки**
```sh
npm run start
```

4. **Откройте в браузере**

5. **Сборка для production**
```sh
npm run build
```

## Используемые технологии

- **Frontend:** React
- **Базовая вёрстка:** HTML/CSS
- **Сборка:** React-scripts
